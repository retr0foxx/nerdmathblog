Exercise 1.6.1. Show that $(0, 1)$ is uncountable if and only if $\mathbb{R}$ is uncountable.
The book already gave a bijective function from $(-1, 1)$ to $\mathbb{R}$ on the earlier pages so this is pretty easy. Take the function $f(x) = x/(x^2 - 1)$. It will be proven that $f$ maps $(-1, 1)$ to $\mathbb{R}$ bijectively. The range of this function is already definitely a subset of $\mathbb{R}$ so that's one part done. Take any $r \in \mathbb{R}$, with a little bit of algebra from solving $r = \frac{x}{x^2 - 1}$, you can take $x = \frac{1 - \sqrt{1 + 4r^2}}{2r}$ and see: $$ \begin{align*} f(\frac{1 - \sqrt{1 + 4r^2}}{2r}) &= \frac{\frac{1 - \sqrt{1 + 4r^2}}{2r}}{(\frac{1 - \sqrt{1 + 4r^2}}{2r})^2 - 1} \\ &= \frac{1 - \sqrt{1 + 4r^2}}{(\frac{(1 - \sqrt{1 + 4r^2})^2}{2r}) - 2r} \\ &= \frac{1 - \sqrt{1 + 4r^2}}{\frac{(1 - \sqrt{1 + 4r^2})^2 - 4r^2}{2r}} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{(1 - \sqrt{1 + 4r^2})^2 - 4r^2} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{1 - 2\sqrt{1 + 4r^2} + 1 + 4r^2 - 4r^2} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{2 - 2\sqrt{1 + 4r^2}} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{2(1 - \sqrt{1 + 4r^2})} \\ &= r\\ \end{align*} $$ Now to prove that $x \in (-1, -1)$. The idea is that, when $r$ is zero, the expression is also zero which is in the interval. While when $r$ is non-zero, the square root is strictly less than 1 + |2r| (which is not true if r is zero) while also being strictly higher than |2r|. That would imply that the final expression is in the interval. Here's the complete proof:
Lemma 1: Suppose $a > 0$, then $\sqrt{a + b^2} = \sqrt{a} + |b|$ if and only if $b = 0$.
Proof: $\sqrt{a + b^2} = \sqrt{a} + |b|$ imply $a + b^2 = a + 2\sqrt{a}|b| + b^2$ which imply $\sqrt{a}|b| = 0$. Since $\sqrt{a}$ is non-zero, then $|b|$ must be zero which also mean $b = 0$. If $b = 0$, then: $$\sqrt{a + b^2} = \sqrt{a + 0} = \sqrt{a} + 0 = \sqrt{a} + |b|$$
Lemma 2: Suppose $a > 0$ and $b \neq 0$, then $\sqrt{a + b^2} < \sqrt{a} + |b|$
Proof: The statement imply $2\sqrt{a}|b| > 0$ therefore $a + b^2 < a + 2\sqrt{a}|b| + b^2 = \left( a + |b| \right)^2$ which imply $\sqrt{a + b^2} < a + |b|$

When $r = 0$, $\frac{1 - \sqrt{1 + 4r^2}}{2r} = 0/0$, taking the limit of the expression, maybe by using L'Hopital or something, it is equal to 0. when $r \neq 0$, by lemma 2 (setting a = 1), it is known that $\sqrt{1 + 4r^2} < 1 + |2r|$ but it is also known that $|2r| < \sqrt{1 + 4r^2}$. Therefore, you can now do: $$ \begin{align*} |2r| < \sqrt{1 + 4r^2} < 1 + |2r| &\implies |2r| - 1 < \sqrt{1 + 4r^2} - 1 < |2r| && \text{(substract by 1)} \\ &\implies 1 - |2r| > 1 - \sqrt{1 + 4r^2} > -|2r| && \text{(negate)} \\ &\implies \frac{1}{|2r|} - 1 > \frac{1 - \sqrt{1 + 4r^2}}{|2r|} > -1 && \text{(divide by $|2r|$)} \\ &\implies 0 > \frac{1 - \sqrt{1 + 4r^2}}{|2r|} > -1 && \text{(for the positive case)} \\ &\implies 0 < \frac{1 - \sqrt{1 + 4r^2}}{-|2r|} < 1 && \text{(for the negative case)} \\ &\implies -1 < \frac{1 - \sqrt{1 + 4r^2}}{2r} < 1 \\ \end{align*} $$ Therefore the expression is always in the interval $(-1, 1)$

In this exercise, I want to highlight a generalization of the template of a diagonalization argument that I learned from a YouTube video. Here's the diagram in the video: $$ \xymatrix{ A^2 \ar[r]^f & B \ar[d]^h \\ A \ar[u]^g \ar[r]_s & B } $$ Now to explain this diagram. So, in general, the shared theme in diagonalization proofs is that you start with an indexed collection of functions, say functions $f_i : A \to B$ for any $i \in A$. And then you would try to create the function $s: A \to B$ that is not already in that indexed collection of functions. It does this by utilizing the indexing set of the collection is the same as domain of each of the functions in the collection which allows you to do a process called we know as diagonalization. So you can create $s$ by making it not equal to any $f_i$ by simply making the $s(i)$ not equal to $f_i(i)$. And so in the process of defining $s$ on all of it's domain, you are essentially going through the process of getting the value of $f_i(i)$, and then creating a value that is not equal to that. In the diagram, the function which generates this not-equal value is $h$, and the function which goes from $i$ to $f_i(i)$ is $g$. This diagram is just showing that $s$ is a composition of $g$, $f$, and $h$ IF you redefine $f_a(b)$ to be $f(a, b)$. Therefore, $s(i) = h(f(g(i)))$. Basically, this is an attempt to generalize diagonalization arguments using category theory which is what the video is actually about. It is an attempt at an introduction to category theory.

Anyway, for the actual proof of Cantor's Theorem. Start with a set $H$ and prove that any mapping from $H$ to $P(H)$ is not onto. Start by defining $k: H \to P(H)$ and assume that it is an onto mapping from $H$ to $P(H)$. By using the diagonalization proof template from before, you can define $A = H$, $B = \{ 0, 1 \}$. And then define $f: A^2 \to B$ (which is just $f: H^2 \to \{ 0, 1 \}$) which, if called with $f(a, b)$, represents whether or not $b$ is in $k(a)$. This means, if you lock $a$ to be some element of $H$, then $f(a, b)$ completely represents $k(b)$ because $k(b) = \{ h | f(a, h) = 1 \}$. The point is that $f(a, b)$ with $a$ being locked is able to represent any subset of $H$. The final goal is to create a set $S \in P(H)$ that $k$ have not mapped any element of $H$ to. This is equivalent to creating a function $s: A \to B$ that is not equal to any $f(a, x)$ as a function of $x$ with $a$ being locked to any element of H and then you can define $S = {b | s(b) = 1}$ To do this, you create the function $g$ which does the diagonalization, therefore $g(i) = (i, i)$, and then create the function $h: B \to B$ which generates the "not-equal" value therefore $h(0) = 1$ and $h(1) = 0$. Then you define $s(i)$ to be $h(f(g(i)))$ which is the same as $h(f(i, i))$. The set that $s$ represents, S, cannot be equal to any $k(b)$. Because, suppose $b \in k(b)$, then, by definition, $f(b, b) = 1$ and so $s(b) = h(f(b, b)) = h(1) = 0$ and so $b$ is not in $S$ therefore $S \neq k(b)$. You can use basically the same logic for if $b \in k(b)$.
