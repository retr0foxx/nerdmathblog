Exercise 1.6.1. Show that $(0, 1)$ is uncountable if and only if $\mathbb{R}$ is uncountable.
The book already gave a bijective function from $(-1, 1)$ to $\mathbb{R}$ on the earlier pages so this is pretty easy. Take the function $f(x) = x/(x^2 - 1)$. It will be proven that $f$ maps $(-1, 1)$ to $\mathbb{R}$ bijectively. The range of this function is already definitely a subset of $\mathbb{R}$ so that's one part done. Take any $r \in \mathbb{R}$, with a little bit of algebra from solving $r = \frac{x}{x^2 - 1}$, you can take $x = \frac{1 - \sqrt{1 + 4r^2}}{2r}$ and see: $$ \begin{align*} f(\frac{1 - \sqrt{1 + 4r^2}}{2r}) &= \frac{\frac{1 - \sqrt{1 + 4r^2}}{2r}}{(\frac{1 - \sqrt{1 + 4r^2}}{2r})^2 - 1} \\ &= \frac{1 - \sqrt{1 + 4r^2}}{(\frac{(1 - \sqrt{1 + 4r^2})^2}{2r}) - 2r} \\ &= \frac{1 - \sqrt{1 + 4r^2}}{\frac{(1 - \sqrt{1 + 4r^2})^2 - 4r^2}{2r}} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{(1 - \sqrt{1 + 4r^2})^2 - 4r^2} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{1 - 2\sqrt{1 + 4r^2} + 1 + 4r^2 - 4r^2} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{2 - 2\sqrt{1 + 4r^2}} \\ &= \frac{2r(1 - \sqrt{1 + 4r^2})}{2(1 - \sqrt{1 + 4r^2})} \\ &= r\\ \end{align*} $$ Now to prove that $x \in (-1, -1)$. sqrt(4r^2) = |2r| sqrt(1 + 4r^2) > |2r| > 0 -> (1) 1 - sqrt(1 + 4r^2) > 1 - |2r| (1 - sqrt(1 + 4r^2))/(2r) > (1 - |2r|)/(2r) (1 - sqrt(1 + 4r^2))/(2r) > (1 - |2r|)/(2r) if r is positive: (1 - sqrt(1 + 4r^2))/|2r| > 1/|2r| - 1 1/|2r| - sqrt(1 + 4r^2)/|2r| > 1/|2r| - 1 -sqrt(1 + 4r^2)/|2r| > -1 by (1), it also means: 0 > -sqrt(1 + 4r^2)/|2r| > -1 0 > 1/|2r| > (1 - sqrt(1 + 4r^2))/|2r| > -1 + 1/|2r| > -1 Therefore 0 > (1 - sqrt(1 + 4r^2))/|2r| > -1
