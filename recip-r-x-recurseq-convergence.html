Regarding sequences in the form of \(x_{n+1} = \frac{1}{r-x_n}\) with \(|r| \geq 2\)

Before I go into such sequences, I will first make one thing clear

Suppose \(a < x < b\), then:

  1. \(\frac{1}{a} < x < \frac{1}{b}\) if and only if \(a < 0 < b\)
  2. \(\frac{1}{a} > x > \frac{1}{b}\) if and only if \(a < b < 0\) or \(0 < a < b\). In other words, they are of the same sign.

Now for the actual sequence. On which subsets of \(\mathbb{R}\) for \(x_n\) does it converge on depending on \(r\)? What are it’s limit points? We will limit our view to \(|r| \geq 2\) for reasons that will be clear later. What are it’s limit points in that case?

Constant points of \(\frac{1}{r - x}\)

To find the constant points of the function, which should intuitively translate to the limit points of the sequence, you can simply try to solve \(x = \frac{1}{r - x}\) for x. Of course, an obvious concern with trying to find the limit points like this is that this means we are assuming that such a value exists. We will not concern ourselves with this for now, we will only intuitively think about why this is valid by simply verifying that the resulting value for \(x\) is algebraically valid and satisfy the original equation.

By using the quadratic equation, you can conclude that \(x = \frac{r \pm \sqrt{r^2 - 4}}{2}\). We will attempt to verify that this is value satisfies the original equation. The most straightforward way to do this is by literally doing \(\frac{1}{1 - x}\) and seeing if it is equal to \(x\). \[ \begin{align*} x &= \frac{r \pm \sqrt{r^2 - 4}}{2} \\ r - x &= \frac{r \mp \sqrt{r^2 - 4}}{2} \\ \frac{1}{r - x} &= \frac{2}{r \mp \sqrt{r^2 - 4}} \\ &= \frac{2}{r \mp \sqrt{r^2 - 4}} \cdot \frac{r \pm \sqrt{r^2 - 4}}{r \pm \sqrt{r^2 - 4}} \\ &= \frac{2 r \pm \sqrt{r^2 - 4}}{(r^2 - (r^2 - 4))} \\ &= \frac{2 r \pm \sqrt{r^2 - 4}}{4} \\ &= \frac{r \pm \sqrt{r^2 - 4}}{2} \\ &= x \end{align*} \] and so, yes, such an \(x\) really does satisfy the original equation. We now intuitively know the constant points of the sequence function which should intuitively translate to limit points of the sequence. Now it’s clear that the reason we used \(|r| \geq 2\) earlier is that, if it less than two, then based on this formula, the limit points will be undefined. Now we will actually start investigating the convergence of this sequence based on the starting point \(x_n\).
