Definition (Open Map). \(f: A \to \mathbb{R}\) is an open map if \(f\) maps open sets in \(A\) into open sets.

Proposition: Suppose \(a < b\) is defined with \(a \in \mathbb{R} \cup \{ -\infty \}, b \in \mathbb{R}\) and \(f: (a, b) \to \mathbb{R}\) is increasing and is an open map and \(f((a, b)) \subseteq (a, b)\) and \(x \in (a, b)\) implies \(f(x) > x\), then if \((x_n)\) is a sequence which starts out in \((a, b)\) and is recursively defined with \(f\), therefore \(x_1 \in (a, b)\) and \(x_{n+1} = f(x_n)\), then \((x_n) \to b\).

Proof. The strategy is to prove that if \((x_n) \to x\) with \(x \neq b\), then \(x \in (a, b)\) must be true. Now since \((x_n)\) is increasing, then it gets arbitrarily close to \(x\) from the left side and since \(f\) is increasing, it preserves inequalities. This means all of \((x_n)\) is less than \(x\) and so everything less than \(x\) stays being less than \(x\) under \(f\). Since \(f\) is strictly increasing, then \(f(x) > x\). Notice the issue here, anything less than \(x\) stays less than \(x\) yet when a value is \(x\) it suddenly jumps to being greater than \(x\). This means any open interval which contains \(x\) will result in a non-open interval under \(f\).

  • \(x_n \in (a, b)\) for all natural \(n\) and \((x_n)\) is strictly increasing
    \(x_1 \in (a, b)\), \(x_{n+1} = f(x_n)\) and \(f((a, b)) \subseteq (a, b)\) clearly implies \(x_n \in (a, b)\) for all natural \(n\).
    \(x < f(x)\) clearly implies \(x_n < f(x_n) = x_{n+1}\) for all natural \(n\).
  • \(a < \lim (x_n) \leq b\)
    Since \((x_n)\) is increasing, then \(x_1 \leq x_n\) for all natural \(n\) and since \(x_n < b\) is also true for all natural \(n\), then by the order limit theorems, \(x_1 \leq \lim x_n \leq b\). Since \(a < x_1\), then \(a < \lim x_n \leq b\).
  • Define \(x = \lim x_n\) and suppose \(x \neq b\), the previous points imply that \(x \in (a, b)\).
  • \(x_n < x\) for all natural \(n\).
    This is mainly because \(f(p) > p\) for any \(p \in (a, b)\). I don’t think I need to explain much further, but otherwise, if there’s an \(n\) where \(x_n \geq x\), then \(f(x_n) = x_{n+1} > x_n \geq x\) and so by induction for every \(m > n\) it follows that \(x_m > x\) which, by the algebraic limit theorems, means that \(\lim (x_m) > x\).
  • For all \(\epsilon > 0\), there exists an \(n\) where \(x - \epsilon < x_n < x\).
    Since \((x_n) \to x\), then for all \(\epsilon > 0\) there exists an \(n\) such that \(|x_n - x| < \epsilon\) which by the fact that \(x_n < x\), then \(x - \epsilon < x_n < x\).
  • For all \(k < x\), it follows that \(f(k) < x\)
    From the previous bullet point, for all \(k < x\), you can pick \(\epsilon = x - k\) and know that there exists an \(n\) where \(k < x_n < x\) therefore, since \(f\) is increasing, then \(f(k) < f(x_n) = x_{n+1}\) and since \(x_{n+1} < x\) then \(f(k) < x_{n+1} < x\) and so \(f(k) < x\).
  • \(f((a, x)) \subseteq (a, x)\) and it an open set.
    It is an open set since \(f\) is an open map. Now, for any \(k \in (a, x)\), we know that \(f(k) < x\) and since \(f((a, b)) \subseteq (a, b)\) in our definitions, then \(f(k) \in (a, x)\).
  • \(f([x, b)) \subseteq [f(x), b)\) and it is not an open set.
    Anything in \((x, b)\) is greater than \(x\) and since \(f\) is increasing then \(k \in (x, b)\) imply \(f(k) > f(x)\) therefore \(f((x, b)) \subseteq (f(x), b)\). Simply unionize \(x\) into the input and \(f(x)\) into the output and you get the desired result.
  • \((a, x)\) is disjoint with \([f(x), b)\).
    \(f(x) > x\) therefore anything in \([f(x), b)\) is strictly greater than everything in \((a, x)\) and anything in \((a, x)\) is strictly less than everything in \([f(x), b)\).
  • This is where the contradiction appears. \(f((a, b)) = f((a, x)) \cup f([x, b))\). \(f((a, x))\) is open and \(f([x, b))\) is not open and the two of them are known to be disjoint. The union of disjoint sets where one of them is not an open set results is a non open set which is a contradiction to the fact that \(f\) is an open map.

Corollary. If, instead, \(f(x) < x\) and \(a \in \mathbb{R}, b \in \mathbb{R} \cup \{ \infty \}\), then \((x_n) \to a\).

Proof.

  • Define \(g(x) = -f(-x)\), \(y_1 = -x_1\), and \(y_{n+1} = g(y_n)\) therefore \(y_1 \in (-b, -a)\)
  • \(g(x)\) is increasing, is an open map, and \(x < g(x)\) in \((-b, -a)\)
    • Clearly, \(x \in (-b, -a)\) implies \(g(x) = -f(-x) \in (-b, -a)\) since \(f((a, b)) \subseteq (a, b)\)
    • Take \(x_1 < x_2\), then \(-x_1 > -x_2\) and since \(f\) is increasing, then \(f(-x_1) > f(-x_2)\) which means \(g(x_1) < g(x_2)\).
    • \(-x\) and \(f(x)\) are open maps and composition of open maps are open maps
  • \((y_n) \to -a\)
    This is implied by the previous theorem since \((y_n)\) and \(g\) satisfies all of the requirements.
  • \(x_n = -y_n\) for all natural \(n\).
    It’s true for \(y_1\) by definition. If \(x_n = -y_n\) for any \(n\), then \(x_{n+1} = f(x_n) = f(-y_n) = -g(y_n) = -y_{n+1}\). By induction, it is true.
  • \((x_n) \to a\)
    Previous bullet point implies \(\lim x_n = -\lim y_n\) by the algebraic limit theorem and \(\lim y_n = -a\) therefore \(\lim x_n = a\).

These problems came straight out of my analysis book

  2. \(x_1 = \sqrt{2}\) and \(x_{n+1} = \sqrt{2 + x_n}\), then \((x_n) \to 2\)
  3. \(x_1 = \sqrt{2}\), \(x_{n+1} = \sqrt{2x_{n-1}}\) then \((x_n) \to 2\)

Lemma (Composition of Open Maps are Open Maps). Suppose \(f: A \to \mathbb{R}\) and \(g: B \to \mathbb{R}\) are open maps with \(g(B) \subseteq A\), then the composition of \(f\) and \(g\) denoted \(h(x) = f(g(x))\) is also an open map.

List of obviously open maps:

  1. \(f(x) = \frac{1}{x}\) is open in \(A\) if everything in \(A\) is of the same sign and \(A\) does not contain zero.
    In other words, \(A = A \cap (0, \infty)\) or \(A = A \cap (-\infty, 0)\).
  2. \(f(x) = ax + b\) is an open map in \(\mathbb{R}\) if \(a \neq 0\).
  3. \(f(x) = \sqrt{x}\) is an open map for all positive real numbers.

With the previous lemma and list of open maps, it should follow that:

  1. \(h(x) = \frac{1}{r - x}\) is an open map in \((-\infty, r)\).
    This is because it’s a composition of \(f(x) = \frac{1}{x}\) and \(g(x) = r - x\). \(g\) maps \((-\infty, r)\) to \((0, \infty)\) and we know that \(f\) is open in \((0, \infty)\).
  2. \(f(x) = \sqrt{2 + x}\) is an open map in \((-2, \infty)\).
  3. \(f(x) = \sqrt{2x}\) is an open map in \((0, \infty)\).

Now to prove that the other requirements of the domain being mapped onto itself with all three functions using algebraic methods

  1. I have talked a lot about number one in another page including the fact that \(f\) maps to itself on \((0, k)\). It has also been proven there that if \(j = \frac{r - \sqrt{r^2 - 4}}{2}\), then \(f\) is increasing in \((j, k)\).
  2. \(f(x) = \sqrt{2 + x}\) maps \((0, 2)\) into itself.
    This is because additions and square roots preserves inequalities and \(0 < x < 2\) imply \(2 < x + 2 < 4\) which means \(\sqrt{2} < \sqrt{x + 2} < 2\) and so the function does map the \((0, 2)\) into itself.
    Then, since \(x < 2\), it’s true that \(x - 1 < 1\) then combining them together implies \(x(x - 1) < 2\) therefore \(x^2 - x < 2\) and so \(x < \sqrt{2 + x}\).
  3. \(f(x) = \sqrt{2x}\) maps \((0, 2)\) into itself.
    Similarly, this is because square roots and multiplications preserves inequalities and \(0 < x < 2\) imply \(0 < 2x < 4\) and so \(0 < \sqrt{2x} < 2\). Again, we will also just accept that \(x < \sqrt{2x}\) for all \(x \in (0, 2)\).
    Then, since \(x < 2\), it’s true that \(x - 2 < 0\) therefore \(x(x - 2) < 0\) therefore \(x^2 - 2x < 0\) meaning \(x < \sqrt{2x}\).

Since all of the functions and sequence starting points satisfy all the requirements for the main proposition earlier, then all of the three statements are true. It’s clear that \(\sqrt{2} \in (0, 2)\) as well, right, which is needed for the second and third statements.
