Exercise 2.2.2. Verify, using the definition of convergence of a sequence, that the following sequences converge to the proposed limit.
(a) $\lim \frac{2n + 1}{5n + 4} = \frac{2}{5}$
(b) $\lim \frac{2n^2}{n^3 + 3} = 0$
(c) $\lim \frac{\sin({n^2})}{\sqrt[3]{n}} = 0$

So the strategy for doing these in this case is to find, for some $n$, the difference (or in other words the $\epsilon$) between that term and the proposed limit which is the opposite of what you'd normally do. Then you can somehow inverse that so that you can then get the $n$ for a certain $\epsilon$ or something. When doing these limit exercises, the nth term will usually always be denoted $a_n$ and the proposed limit $a$
(a) $$ \begin{align} \frac{2}{5} - a_n &= \frac{2}{5} - \frac{2n + 1}{5n + 4} \\ &= \frac{2}{5} - \left( \frac{2}{5} + \frac{2n + 1 - (5n + 4) \cdot \frac{2}{5}}{5n + 4} \right) \\ &= \frac{2}{5} - \left( \frac{2}{5} + \frac{1 - \frac{8}{5}}{5n + 4} \right) \\ &= \frac{2}{5} - \left( \frac{2}{5} - \frac{\frac{3}{5}}{5n + 4} \right) \\ &= \frac{2}{5} - \left( \frac{2}{5} - \frac{3}{25n + 20} \right) \\ |a - a_n| &= \frac{3}{25n + 20} \\ \end{align} $$ Now we know that the epsilon needs to atleast be that value when you choose such an $N$. The strategy now is to start with some epsilon, choose an $N$ which would be less than epsilon by turning epsilon into a smaller rational number which is in the form that we had earlier, then prove that any $n \geq N$ is also smaller than that.
Take some $\epsilon \geq 0$, then we know there exists an $N$ such that $\frac{1}{N} < \frac{\epsilon}{3}$ therefore $\epsilon > \frac{3}{N} > \frac{3}{25N + 20} = |a_N - a|$. For all $n \geq N$, it is clear that $\frac{3}{25n + 20} \leq \frac{3}{25N + 20}$ therefore $|a_n - a| \leq |a_N - a| < \epsilon$. End of proof.

(2) Actually, there's also the strategy of constructing a larger-valued expression that is easier to work with for the difference between the nth term compared to the proposed limit. In this case, we know that $$ |a_n - a| = \frac{2n^2}{n^3 + 3} < \frac{2n^2}{n^3} = \frac{2}{n} $$ and so $|a_n - a| < \frac{2}{n}$. Take any $\epsilon > 0$, then there exists some $N$ such that $\frac{1}{N} < \frac{\epsilon}{2}$ and so $\frac{2}{N} < \epsilon$. therefore $|a_N - a| < \epsilon$. For any $n \geq N$, we know that $|a_n - a| < \frac{2}{n} < \frac{2}{N} < \epsilon$ therefore $|a_n - a| < \epsilon$. End of proof.
(c) First of all, it is known that for any x, $|\sin(x)| \leq 1$ and so $|a_n - a| = |a_n| = \left| \frac{\sin(n^2)}{\sqrt[3]{n}} \right| \leq \frac{1}{\sqrt[3]{n}}$. Take any $\epsilon > 0$, then it is known that there exists an $N \in \mathbb{N}$ such that $\frac{1}{N} < \epsilon^3$ and so $\frac{1}{\sqrt[3]{N}} < \epsilon$ therefore $|a_N - a| < \epsilon$. For any $n \geq N$, it is known that $|a_n - a| = \frac{1}{\sqrt[3]{n}} < \frac{1}{\sqrt[3]{N}} < \epsilon$ and so $|a_n - a| < \epsilon$. End of proof.
