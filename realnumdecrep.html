All real numbers have a decimal representation

The goal here is to prove that every real number have a decimal representation. A decimal representation of a real number r is defined as a sequence $r_n$ $\forall n \in \mathbb{Z}$ such that $\sum_{i=-\infty}^{\infty} r_i10^i = r$.
The strategy for the proof is to start with any number $a$, and then create a decimal representation and prove that the decimal representation actually represents $a$. It is obvious that if you can have a decimal representation of a positive number, then you can have a decimal representation of any negative numbers. Therefore, the first step of the proof will only prove the positive case. The first step is to define $r_n$, $p_n$, and $d_n$ for every natural number n. $r_1$ is defined to be $r_1 = |a|$, then $r_{n+1} = r_n - d_n10^{p_n}$, while the rest of the variables is defined as: $$ p_n = \begin {cases} -\infty, & r_n = 0 \\ \max \{p | p \in \mathbb{Z} \land 10^p \leq r_n\}, & r_n \neq 0 \end {cases} $$ $$ d_n = \begin{cases} 0, & r_n = 0 \\ \max \{d | d \in \{1, 2, ..., 9\} \land d \cdot 10^{p_n} \leq r_n \}, & r_n \neq 0 \end{cases} $$ The next step is to prove that these steps work. Which is needed because both of these might take the maximum of a set and that does not always exist. First is to prove that there, for any real number, there is a power of 10 less than it:
$\forall \epsilon > 0$ $\exists n \in \mathbb{N}: \frac{1}{n} \leq \epsilon$ by the Archimedean property $\implies \frac{1}{10^n} < \frac{1}{n} \leq {\epsilon} \land \frac{1}{10^n} = 10^{-n} \implies \forall \epsilon > 0$ $\exists n \in \mathbb{Z}: 10^n \leq \epsilon$
Next is to prove $r_n \geq 0$ $\forall n \in \mathbb{N}$ by using induction:
1. $r_1 \geq 0.$
2. $r_n \geq 0 \implies r_n = 0 \lor r_n > 0$
2. a. $r_n = 0 \implies d_n = 0 \implies r_{n + 1} = r_n - 0 \cdot 10^{p_m} = r_n - 0 = 0 - 0 = 0$
2. b. $r_n > 0 \implies p_n = \max \{p | p \in \mathbb{Z} \land 10^p \leq r_n\} \implies p_n$ exists because that set is non-empty as we've proven earlier $ \implies 1 \in \{d | d \in \{1, 2, ..., 9\} \land d \cdot 10^{p_n} \leq r_n \} \implies d_n $ exists because there is atleast 1 element in the set in the definition of $d_n$ $\implies r_{n+1} = r_n - d_n10^{p_n} \geq 0$ by the definition of $d_n$ and $p_n$ when $r_n \geq 0$
2. a $\land$ b $\implies \left(r_n \geq 0 \implies r_{n+1} \geq 0 \right)$
By induction, we can conclude that $r_n \geq 0$ $\forall n \in \mathbb{N}$. Notice that we have also proved that $d_n$ and $p_n$ is always defined for all natural number n.

The next step is to create the sequence to represent $|a|$ to be: $$ a_n = \begin{cases} d_{\min \{m | p_m = n\}}, & \text{if } n \in \{p_m | m \in \mathbb{N}\} \\ 0, & otherwise \end{cases} $$ This works because $n \in \{p_m | m \in \mathbb{N}\} \implies \exists m \in \mathbb{N}: p_m = n \implies \{m | p_m = n\} \neq \emptyset \implies \min \{m | p_m = n\}$ exists.

And so the next big step is to prove that $a_n$ really does represent $|a|$. The strategy is to first prove that $\lim_{n \to \infty} r_n = 0$, and then maybe that $\sum_{i=1}^{\infty} d_i10^{p_i} = |a|$ which is an intuitive implication of the former. The first thing we'll do is to prove $\lim_{n \to \infty} r_n = 0$ which can be done by proving that $\forall \epsilon > 0$ $\exists N \in \mathbb{N}: \forall n > N, r_n < \epsilon$. The strategy to prove this is to first prove the implication $(r_m \neq 0 \implies r_{m + 1} < 10^{p_m})$

Take any $m \in \mathbb{N}$ such that $r_m \neq 0$ then we know that $d_m10^{p_m} \leq r_m$. From the definition of $d_m$, $d_m + 1$ satisfies $d_m + 1 \notin \{1, 2, ..., 9\} \lor (d_m + 1)10^{p_m} > r_m$. If it satisfies $d_m + 1 \notin \{1, 2, ..., 9\}$, then since $d_m \in \{1, 2, ..., 9\}$, then $d_m + 1$ must be 10 $\implies (d_m + 1)10^{p_m} = 10^{p_m + 1}$ and based on the definition of $p_m$, it is known that $10^{p_m + 1} > r_m$ $\implies (d_m + 1)10^{p_m} > r_m$. The next step is to prove that the actual limit. Start with any $\epsilon > 0$, then $\exists k \in \mathbb{Z}$ such that $10^k < \epsilon$ which is something that has been proved earlier.
