The Cauchy Criterion

Definition 2.6.1 (Cauchy Sequence). A sequence $(a_n)$ is Cauchy if for every $\epsilon > 0$, there exists an $N \in \mathbb{N}$ such that whenever $m, n \geq N$ it follows that $|a_n - a_m| < \epsilon$.

This definition is very similar to the definition of a convergent sequence, the only difference is that in here the terms get arbitrarily closer to each other while in sequences that converge the term get arbitrarily closer to it's limit.

Theorem 2.6.2. Every convergent sequence is a Cauchy sequence.

Proof. If for any $\epsilon > 0$, there exists a natural $N$ such that for all $n \geq N$ it follows that $|a_n - a| < \frac{\epsilon}{2}$, which is the case for convergent sequences, then you can use the triangle inequality to conclude that for every $m, n \geq N$, $|a_n - a_m| < \epsilon$. ($|a_n - a| < \epsilon/2$ and $|a_m - a| < \epsilon/2$ combined with the triangle inequality $|a_n - a_m| \leq |a_n - a| + |a_m - a|$ implies $|a_n - a_m| < \epsilon$).

Lemma 2.6.3. Cauchy sequences are bounded.

Proof. Just take any $\epsilon > 0$. We know that there's some $N \in \mathbb{N}$ such that every term $n$ after $N$ in the sequence satisfies $|a_N - a_n| < \epsilon$, therefore only a finite number of terms of the sequence is outside of $(a_N - \epsilon, a_N + \epsilon)$ while the rest is inside, hence it is bounded by $\max \left( \{ |a_N - \epsilon|, |a_N + \epsilon| \} \cup \{a_i | i < N\}| \} \right)$, which exist since the latter set is finite.

Theorem 2.6.4 (Cauchy Criterion). A sequence converges if and only if it is a Cauchy sequence.

Proof. Denote the Cauchy sequence to be $(a_n)$. We have proven earlier that every convergent sequence is a Cauchy sequence. Now if a sequence is a Cauchy sequence, we will prove that it converges. First, take any of it's convergent subsequence. This is possible since it is bounded by the previous lemma and hence contains a convergent subsequence by the Bolzano-Weierstrass theorem.

Suppose this subsequence is represented by the indices $(d_i) \in \mathbb{N}$ and we denote the limit of the subsequence it represents to be $d$. The following will be a proof that the original sequence converges to $d$. First, for any $\epsilon > 0$, there is a natural $N$ such that for all $n, m \geq N$, it follows that $|a_n - a_m| < \epsilon$ because the original sequence is Cauchy. Take any index $M$ such that $d_m \geq N$ and that for all $m \geq M$, $|a_{d_m} - d| < \epsilon$, hence we know that for any $m \geq M$ and $n \geq N$, $|a_n - a_{d_m}| < \epsilon$ while $|a_{d_m} - d| < \epsilon$, therefore $|a_n - d| < 2\epsilon$ by the triangle inequality. Therefore $N$ satisfies for all $n \geq N$ it follows that $|a_n - d| < 2\epsilon$. Replace all of the $\epsilon$s except for the first $\epsilon > 0$ with $\frac{\epsilon}{2}$ and we're done with the proof.
